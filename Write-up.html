*Routed Pages:
                        resource :users -> standard paths from generating with scaffolding
                        resource :session -> only :new, :create, :destory
                        New :Root/signin path : session/new -> localhost:3000
                        create: creates a session and cookie with saved information. (no page)
                        destory: signout path - >Deletes the session and cookie, “Logout” link
                        signup path : users/new -> localhost:3000/signup
                        addcontact path : pages/addcontact -> localhost:3000/addcontact
                        profile path : pages/profile -> localhost:3000/profile
                        search path : pages/search -> localhost:3000/search
                        message path : pages/message -> localhost:3000/message

*Descritpion : An online phone book web application. (Milestone 1)

*Minimun Functionality : Signup for new users needs to be created, along with a login based off information of the user. Once a user signs in, they will be able to add a contact, view/edit/delete their contacts, view/edit their information, and send messages (emails) to a contact. Once the user is finished user just clicks “logout”.
